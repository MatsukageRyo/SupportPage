## 絶対朝活スイッチ / No-Excuses Morning Lock {#no-excuses-morning-lock}

### 概要 / Overview
- JP: 夜にロックを開始し、朝活を規定時間できたら解除する「朝活特化」アプリです。
- EN: A morning-routine app that starts a lock at night and unlocks after you complete the required morning activity time.

### お問い合わせ / Support
- Email: [mat.3.tung@gmail.com](mailto:mat.3.tung@gmail.com)
- GitHub Issues: <https://github.com/MatsukageRyo/FocusOne/issues>

### よくある質問 / FAQ
**JP**
**Q1.** これは何のアプリですか？  
**A1.** 朝活の時間を確保するため、ユーザーが選んだアプリを一時的に制限します。朝活を規定時間続けると解除されます。

**Q2.** 写真ライブラリへのアクセスは必要ですか？  
**A2.** 不要です。本アプリは写真/ファイルへアクセスしません。

**Q3.** どんな許可が必要ですか？  
**A3.** 通知（寝る/起きるリマインド）と、スクリーンタイム（選択したアプリの制限）です。許可しなくても一部機能（記録など）は使えます。

**Q4.** スクリーンタイム制限が効きません  
**A4.** 端末の設定で「スクリーンタイムへのアクセス」を許可し、ブロック対象アプリを選択してください。シミュレータでは動作しない場合があります（実機推奨）。

**Q5.** 収集するデータはありますか？  
**A5.** 目標や時間設定、達成記録などは端末内に保存されます。サーバーへ送信しません（現時点）。

**Q6.** ロック中に困ったら？  
**A6.** 本アプリが制限するのはユーザーが選択したアプリのみです。重要な連絡に必要なアプリをブロック対象に含めないよう注意してください。

**EN**
**Q1.** What is this app?  
**A1.** It helps you protect your morning routine by temporarily restricting selected apps. It unlocks after you complete the required time.

**Q2.** Does it need Photos access?  
**A2.** No. The app does not request access to your photo library or files.

**Q3.** What permissions does it need?  
**A3.** Notifications (reminders) and Screen Time (to restrict selected apps). You can still use some features without permissions.

**Q4.** App blocking doesn't work  
**A4.** Enable Screen Time access in Settings and select apps to restrict. Some features may not work on Simulator; use a real device.

**Q5.** What data do you collect?  
**A5.** Goals, schedules, and logs are stored on device. No server transmission at this time.

**Q6.** What if I need access during a lock?  
**A6.** Only the apps you selected are restricted. Do not include important communication apps in the restricted list.
